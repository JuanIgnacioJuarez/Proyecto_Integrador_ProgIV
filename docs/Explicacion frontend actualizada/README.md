# Explicacion Frontend (actualizada)

Esta carpeta documenta el frontend actual en formato didactico (archivo por archivo), siguiendo el mismo objetivo del material de backend: entender arquitectura y flujo real del proyecto.

## Cobertura

- Se documentaron todos los archivos de frontend/src con extensiones .ts, .tsx y .css.
- Se mantuvo la estructura de carpetas (app, entities, features, pages, shared, widgets) para que navegar sea directo.

## Recomendacion para estudiar

1. Empeza por app/main.tsx y app/App.tsx.
2. Segui por entities/*Context.tsx + entities/*Api.ts.
3. Despues mira pages/ y finalmente features/ y shared/ui/.
